Use WebGL in the no-dimensional world.

Download!

Easy to Use

Zero.js allows you to skip tons of boilerplate codes for using WebGL and to concentrate on writing the essential code to draw the origin.

Lean Enough

There is only one geometry representing the origin, which has no dimension, no direction and no shape.

Familiar

The APIs of zero.js are similar to three.js, which is the de-facto standard library in using WebGL.
